Kotlin est le nouveau langage de programmation de Google et se demarque par sa simplicité du code.
Il est supporté par Android studio et peu potentiellement remplacé Java cependant je pense que Kotlin ne devrait pas 
etre notre choix pour se projet pour les points suivants :

L'avantage de Java c'est que le code est clairement annoncé et plus lisible par rapport a Kotlin qui lui est plus compact
et les erreurs sont moins previsibles

Les differents forum d'entraide sont plus nombreux sur java que sur Kotlin ce qui fait qu'il est plus 
facile de trouver sa reponse aujourd'hui sur java

Java est un langage qui a plus d'ancienneté donc ca fiabilité sur le marché est bien réelle.

Kotlin est plus gourmand car il fait gonfler la taille de l'APK quand l'application est compilé de 800ko 
ce qui peut etre une mauvaise chose
lors du telechargement de l'application
